# WCV3 Viewer public shell

This repository contains only the generic static shell in `public/`. Local packers, source data, passwords, and `.wcv` exports must remain in a separate sibling directory and must never be committed or uploaded.

Run `npm test` and `npm run verify` before publishing. The Pages workflow uses official action major versions because reviewed commit SHAs were not confirmed in this environment; pin each action to a reviewed full commit SHA before production publication.
